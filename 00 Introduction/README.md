# What is R?

R is an open-source programming language and software environment mainly used for:

* Data Analysis
* Statistics
* Data Visualization
* Research
* Machine Learning
* Business Analytics

R was created specifically to work with data, which makes it one of the most popular languages among data analysts, statisticians, researchers, and data scientists.

# Why Learn R?

Learning R provides many benefits:

### Easy to Learn

R has a simple syntax that is beginner-friendly.

### Free and Open Source

Anyone can download and use R without paying any license fees.

### Powerful for Data Analysis

R contains many built-in functions for working with data.

### Excellent Visualizations

R can create professional charts and graphs with just a few lines of code.

### Large Community

Millions of users contribute packages, tutorials, and learning resources.

# Understanding Programming

Programming is simply giving instructions to a computer.

For example:

You tell the computer:

1. Store a number.
2. Add another number.
3. Display the result.

The computer follows these instructions exactly as written.

A collection of such instructions is called a program.

---

# Your First R Code

Let's write the simplest R program.

```r
print("Hello, World!")
```

Output:

```text
[1] "Hello, World!"
```

Congratulations! 🎉

You have written your first R program.

---

# How R Executes Code

When R reads your code:

1. It starts from the first line.
2. Executes one line at a time.
3. Produces the output.
4. Stops when the program finishes.

Example:

```r
print("Welcome")
print("To")
print("R Programming")
```

Output:

```text
[1] "Welcome"
[1] "To"
[1] "R Programming"
```
